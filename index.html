About

This project was inspired by my childhood love for classic snake games. As a passionate developer, I wanted to recreate the nostalgic experience of playing the snake game on modern devices.

Throughout the development process, I enjoyed refining my programming skills and exploring various game mechanics. This project is a portfolio piece created as part of my journey at Holberton School.

For more information about Holberton School, please visit holbertonschool.com.
